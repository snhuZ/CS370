# CS370
I was given starter code that defined the maze environment, neural network model, and experience replay system. My task was to complete the Q training algorithm so the pirate agent could learn to find the treasure. I wrote the training loop that reset the environment, selected actions using exploration and exploitation. I aldo stored theexperiences in memory, and trained the neural network using sampled data. I also tracked the win rate to ensure the model learned a reliable path to the goal.

This project showed me how computer scientists design algorithms and systems that allow computers to solve complex problems. Computer scientists develop software, artificial intelligence, and data systems that impact everyday life. Their work matters because it helps automate tasks, improve efficiency, allows people to focus on other problems.

When approaching a problem, I break it into smaller parts, analyze the existing system, and build a solution step by step. I test and refine the code to ensure it works correctly.
